# Pathway checklist (print & keep)

Open this page and press **Ctrl/Cmd + P** to print a clean one-pager (the sidebar drops
away automatically). Keep it next to you while you film. It's the
[complete pathway](the-pathway.md), condensed to checkboxes.

**Idea & plan**

- [ ] One idea only — I can finish "after 60s the viewer will understand ___"
- [ ] Brief filled: topic, takeaway, audience, hook, platform

**Script**

- [ ] 4 columns: Time | Spoken | On-screen | Visual
- [ ] ~110–150 words, one idea, jargon defined
- [ ] Hook in the first 3 seconds; ends with "follow to learn more"
- [ ] Framed as *how the system works*, never *what you'll earn*

**Voice & visuals**

- [ ] Clean audio (real voice, or AI voice labelled)
- [ ] 4–6 abstract clips/images + a title card, all vertical 9:16
- [ ] No realistic fake people, no fake charts

**Assemble**

- [ ] Captions burned in
- [ ] Visual changes roughly every 2 seconds

**Disclosures — first 3–5 seconds + caption**

- [ ] "Not financial advice" — always
- [ ] "#Ad" / "Sponsored" — if recognized for posting
- [ ] AI label — if realistic AI media

**Post**

- [ ] Pre-publish checklist passed
- [ ] Caption tailored per platform
- [ ] Posted / scheduled

**After**

- [ ] Noted what landed, for next time
