# Pre-Publish Checklist — Run This Every Single Time

Copy it into your notes and tick every box before you post. If any **Compliance** box fails, **do not publish** until fixed.

## ✅ Compliance (mandatory — all must pass)

- [ ] **No investment framing.** Nothing calls DGD an investment or implies financial gain. ([discipline](../compliance/communications-discipline.md))
- [ ] **No price prediction.** No "$X target," "to the moon," "early," charts implying gains.
- [ ] **No return/profit promises.** No "returns," "ROI," "passive income," "I made money."
- [ ] **No solicitation.** CTA is "learn more / read the white paper / follow," never "buy."
- [ ] **Banned-word scan done** on script, captions, on-screen text, thumbnail, hashtags. ([do & don't](../compliance/do-and-dont-language.md))
- [ ] **FTC disclosure present** (if rewarded): spoken in first seconds + on-screen + caption front-loaded + platform branded-content toggle ON. ([FTC](../compliance/ftc-disclosure.md))
- [ ] **AI disclosure present** (if realistic AI voice/avatar/video): platform label + on-screen note. ([AI disclosure](../compliance/ai-disclosure.md))
- [ ] **"Not financial advice"** stated (spoken + caption).
- [ ] **Facts verified** against the white paper / [dgd/](../dgd/glossary.md). No invented numbers.
- [ ] **No realistic fake people/events** generated. No false attribution. ([video gen](../tools/video-generation.md))

## ✅ Quality

- [ ] Hook lands in the **first second** (visual + words). ([hooks](../craft/hooks-library.md))
- [ ] **One idea**, delivered clearly, with a payoff.
- [ ] **Captions** burned in and **proofread** (esp. names/numbers).
- [ ] Visual change every ~2 seconds; no dead air.
- [ ] Audio clean: voice on top, music ducked, no clipping.
- [ ] Strong **first frame** (it's your de-facto thumbnail).
- [ ] Loop-back / soft CTA at the end.

## ✅ Technical

- [ ] **1080×1920, 9:16**, MP4. ([specs](../craft/platform-specs.md))
- [ ] Text inside the **safe zone** (not under platform UI).
- [ ] **No unwanted watermark** from any tool.
- [ ] Music is from a **cleared library** with license saved (and covers this platform). ([music](../tools/music-and-sound.md))
- [ ] Per-platform: title/caption/hashtags tailored; disclosure label toggled.

## ✅ Music licensing
- [ ] Track cleared for **commercial + this platform + short-form**.
- [ ] Attribution added if required.

## The final gut check
> "If a viewer never acquires a single coin, is this still a useful, honest, educational video?"
> If **no** → it's a pitch, not education. Rewrite before posting.
