# Platform Specs & Posting Cheat-Sheet

Sizes, lengths, and posting habits per platform. For *policy* rules see [platform-policies.md](../compliance/platform-policies.md).

## Universal export settings
- **Aspect ratio:** 9:16 vertical
- **Resolution:** 1080 × 1920
- **Frame rate:** 30fps (60fps fine)
- **Format:** MP4 (H.264), AAC audio
- **Safe zone:** keep text/captions in the **center ~80%** — platform UI (captions, buttons, profile) covers the bottom ~15% and right edge.

## Per-platform

| | TikTok | Instagram Reels | YouTube Shorts | X (video) |
|---|---|---|---|---|
| Ideal length | 15–34s (sweet spot); up to longer | 15–30s; up to 90s+ | 15–60s (≤3 min allowed) | 30–60s; concise |
| Aspect | 9:16 | 9:16 | 9:16 (≤60s = Short) | 9:16 or 16:9 |
| Captions | Burn-in + auto | Burn-in + auto | Burn-in + auto | Burn-in (autoplay muted) |
| Hashtags | 3–5 focused | 3–5 + topic | In title/description | 1–3 |
| Cover/thumb | Pick a clean frame | Cover frame | **Custom thumbnail** matters | First frame |
| Disclosure | Branded toggle + in-video | Paid Partnership + in-video (first 3s) | "Paid promotion" + in-video | Paid partnership label |

## Captions & hashtags by platform
- **TikTok:** punchy caption, a question to drive comments, 3–5 niche hashtags (#soundmoney #economics #cryptoeducation). Disclosure label at the **front**.
- **Instagram:** hook in caption first line; Reels need **spoken + on-screen disclosure in first 3s**. Use Paid Partnership tool.
- **YouTube Shorts:** **title is searchable** — write a clear, keyword-true title ("What is sound money? Explained in 45s"). Thumbnail/title drive long-tail discovery.
- **X:** lead the post text with the hook; native video autoplays muted, so on-screen captions are essential. Consider threading context below.

## Posting cadence (sustainable beats sporadic)
- Aim for **3–7 posts/week** if you can sustain it; consistency compounds.
- **Repurpose one video across all four platforms** (tailor caption + disclosure per platform; satisfy the strictest). → [workflows](../tools/workflows.md)
- Best-time-to-post varies; check each platform's analytics and post when *your* audience is active.

## Accessibility = reach
- Captions on always (mute viewing + hearing-impaired + non-native speakers).
- High-contrast text, legible font, ≥ ~1.5s on screen per caption line.
- Describe key visuals in narration where possible.
