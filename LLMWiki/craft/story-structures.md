# Story Structures for 30–60s Explainers

Reusable skeletons. Drop a DGD topic into any of these and you have a script.

## 1. Hook → Gap → Payoff → Loop (the workhorse)
- **Hook (0–1s):** scroll-stopper. → [hooks](hooks-library.md)
- **Gap (1–5s):** name the thing they don't know ("but here's what nobody explains…").
- **Payoff (5–35s):** deliver the insight, one idea, with visuals.
- **Loop (35–45s):** reopen curiosity + soft CTA ("follow for the rest / read the white paper").

*Example (Scarcity):* "Saving money is making you poorer 〔hook〕. We're taught saving is safe — but the supply keeps growing 〔gap〕. The US money supply grew ~7,000% in a century; more units chasing the same goods means each unit buys less. Code can cap supply instead — like a 21-million limit that can't be raised 〔payoff〕. That's one of six things 'real money' needs — follow for the other five 〔loop〕."

## 2. Myth → Reality → Why it matters
- State a common belief → flip it → explain the consequence.
- *Example:* "More coins means more value 〔myth〕 — actually a fixed cap matters more than the number 〔reality〕 — because predictability is what makes money hold value 〔why〕."

## 3. Question → Exploration → Answer
- Open with a real question, explore 2–3 beats, land the answer.
- *Example:* "What actually makes something money? Menger said: the most saleable good wins. Durable, divisible, portable, scarce, wanted. Gold hit all five. Now code can too."

## 4. Number → Breakdown → Takeaway
- Lead with a striking stat, decompose it, leave one lesson.
- *Example:* "4 numbers decide a coin's fair value: adoption — weighted 70% — plus transactions, transaction value, and active developers. Mostly, it's about how many people really use it."

## 5. Then vs. Now (history hook)
- Past → present parallel. Great for monetary history.
- *Example:* "1920s stock market: tons of speculation, no way to value anything. 2026 crypto: same problem. Here's one attempt at a yardstick."

## 6. The Analogy Build
- One sticky analogy, extended.
- *Example:* "A ruler that changes length every day would be useless. That's what inflation does to the dollar as a measure of value."

## 7. Listicle (series engine)
- "N things…" with quick beats; each item can also be its own video.
- *Example:* "6 rules of real money: scarcity, stable pricing, free adoption, decentralized governance, freedom to transact, circulation."

## Structural rules that apply to all
- **One idea per video.** If you have two, make two videos.
- **Front-load value.** Don't save the good part for the end on short-form.
- **Show, don't just say.** Pair each line with a visual change. → [editing](../tools/captions-and-editing.md)
- **End educational.** CTA is "learn more / follow / read the white paper," never "buy." → [discipline](../compliance/communications-discipline.md)

## Map structures to DGD topics
| Topic | Best structure |
|---|---|
| Inflation / why money loses value | Number → Breakdown |
| Six Pillars (series) | Listicle + one Hook→Gap→Payoff per pillar |
| CFV / fair value | Number → Breakdown or Question → Answer |
| "Is it really money?" circulation | Myth → Reality |
| Austrian economists | Then vs. Now / Question → Answer |
| Scarcity / 21M cap | Myth → Reality or Analogy |
