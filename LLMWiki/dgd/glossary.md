# DGD Glossary

Plain-language definitions. Use these to keep your videos accurate. Pull a single term and you have a 20-second explainer.

| Term | Plain meaning |
|---|---|
| **Digital Gold (DGD)** | A Layer-1 cryptocurrency designed as wealth-preserving money; created by John Wright Gotts / Digital Gold Foundation (WP). |
| **Layer-1** | A base blockchain network that runs on its own (e.g., Bitcoin, Ethereum, DGD) rather than on top of another chain. |
| **Austrian economics** | A school of economic thought (Menger, Mises, Hayek) emphasizing sound money, free markets, and money arising from voluntary exchange. |
| **Six Pillars of Perfect Money** | Scarcity, Stable Pricing, Free Adoption, Decentralized Governance, Freedom to Transact, Adequate Circulation (WP §2.2). |
| **DGSB — Digital Gold Standard Benchmark** | A fixed reference point: Bitcoin's measurable fundamentals frozen at Dec 2024 ($1.983T market cap, $100k/coin). It does not change (WP §3). |
| **CFV — Crypto Fair Value** | A formula that values any coin's fundamentals against the DGSB to produce a "Fair Coin Price" (WP §4). |
| **Fair Coin Price** | The price output of the CFV formula for a given coin. |
| **Composite Score** | The weighted blend of the four CFV metrics for a coin (WP §10.3). |
| **The four metrics** | Adoption (70%), Annual Transactions (10%), Annual Transaction Value (10%), Active Developers (10%) (WP §3.2). |
| **PoP — Proof-of-Participation** | DGD's distribution model: coins released continuously as the network grows, split among all current users (WP §5). |
| **`N`** | The current total number of users — the single variable that drives DGD's price and supply (WP §5). |
| **Validation** | Acquiring DGD by pre-loading funds and claiming your per-user share as the network grows (WP §10.1). |
| **DigitalGoldX** | The web account/platform that serves as the interface for validation (WP §5). |
| **QT wallet** | The wallet only the participant controls, where validated coins are delivered (WP §5). |
| **Single-price architecture** | No fluctuating bid/ask; one stable price, enforced via cooperating-venue agreements, so the coin can circulate (WP §8). |
| **Cooperating-venue exclusivity** | Agreements with trading venues that support the single-price commitment (WP §8). |
| **Supply-chain circulation** | The idea that real money flows through the whole chain (merchant→wholesaler→…→raw material), not just consumer→merchant (WP §9). |
| **Participation pathways** | The four ways to participate: Validation, Referrals, Promotional, Supply-chain onboarding (WP §10). |
| **Promotional participation** | The pathway that rewards public educational content about DGD — the ambassador pathway (WP §10.1). |
| **Proof-of-Work (PoW)** | Consensus by spending computing power to secure the network (Bitcoin's method). |
| **Proof-of-Stake (PoS)** | Consensus by committing held coins to validate the network. DGD uses a hybrid PoW/PoS (WP). |
| **SegWit (Segregated Witness)** | An efficiency upgrade (from Bitcoin) that DGD's codebase incorporates (WP). |
| **Fee burning** | Destroying transaction fees rather than paying them to anyone — so no party profits from operating the network (WP §12). |
| **Tor V3 onion** | Privacy networking DGD supports natively for anonymous, encrypted transactions (WP §2.2). |
| **Howey test** | The US legal test for whether something is an "investment contract"/security. DGD is designed to fall outside it (WP §12). |
| **Digital commodity** | A 2026 crypto-asset category for tokens whose value comes from protocol operation + supply/demand "rather than from expectations of profit" — **not** a security. DGD is designed to fit it (WP §12.7). |
| **Atkins Safe Harbor** | A proposed 2026 SEC framework ("Regulation Crypto Assets") with exemptions and an investment-contract safe harbor; DGD is designed to align with its principles (WP §12.8). A **legal** term, not "safe investment." → [safe-harbor page](positioning-safe-harbor.md) |
| **Safe harbor (coin)** | Shorthand for "designed to stay outside securities law." ⚠️ Does **not** mean safe investment / safe haven. |
| **Degen** | Crypto slang for a high-risk speculator/gambler. DGD is the **anti-degen**: "not a speculative instrument" (WP Abstract). A culture to *speak to*, not a play to *sell*. → [positioning](../craft/positioning-and-audiences.md) |
| **Rekt** | Crypto slang for being wiped out by a bad trade. Useful for contrast content (sound money vs gambling). |
| **Store of value** | An asset held to preserve wealth (e.g., gold, Bitcoin). The WP frames Bitcoin as store-of-value "digital gold" and DGD as circulating "wealth-preserving money" — complementary roles (WP §17). |
| **Digital Gold Foundation** | The non-profit that received the donated coin and funds industry-wide monetary-reform work (WP). |

## Terms ambassadors should AVOID (compliance)
"Investment," "returns," "ROI," "moon," "get in early," "guaranteed," "passive income," "to the moon," "price prediction." → see [do & don't language](../compliance/do-and-dont-language.md).
