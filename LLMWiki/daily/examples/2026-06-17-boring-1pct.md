# Example Video — "The Boring 1%"

> A complete, ready-to-shoot package built from today's [daily drop](../2026-06-17.md), Idea 3. **Educational only** — sound money vs. the casino, never a "play." Built to be made with free AI tools ([Veo 3.1](../../tools/video-generation.md) for b-roll, free TTS for voice, CapCut for captions). Run the [pre-publish checklist](../../templates/pre-publish-checklist.md) before posting.

- **Format:** vertical 9:16, ~38 seconds, ~110 spoken words
- **Platforms:** TikTok / Reels / Shorts / X
- **Structure:** Hook (≤1s) → contrast → payoff (one idea) → loop-back CTA
- **Angle:** DGD is the *anti-degen* coin — "boring on purpose," nothing to gamble on by design

## ⚠️ Required disclosures (do not skip)
- **FTC sponsorship — front-loaded, spoken + on-screen in the first ~3s:** "Heads up: I'm in Digital Gold's ambassador program, so this is a sponsored educational video." → [FTC disclosure](../../compliance/ftc-disclosure.md)
- **AI label:** the b-roll is AI-generated, so apply each platform's AI/synthetic-media label + an on-screen "AI-generated visuals" tag. NY's synthetic-performer rule is in effect as of 2026-06-09. → [AI disclosure](../../compliance/ai-disclosure.md)

## ✅ Compliance notes (why this passes)
- No "invest / returns / moon / 100x / early / buy / pump." Degen lingo ("casino," "bet," "ape") is used only to **contrast and hook**, never to sell a play. → [do & don't](../../compliance/do-and-dont-language.md)
- No price prediction; the steady graph illustrates design, not a forecast.
- Facts are design/mechanics only and trace to approved talking points: fixed supply & rules, no staking rewards, fees burned, value from the protocol "rather than from expectations of profit." → [approved talking points](../../dgd/approved-talking-points.md)
- CTA is "read the white paper / follow" — never "buy."
- Disclosures front-loaded (FTC) + AI label applied.
