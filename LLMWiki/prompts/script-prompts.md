# Prompt Library — Scripts & Ideas

Copy-paste prompts for AI writers (ChatGPT, Claude, Gemini). **Every prompt embeds the compliance guardrails** so the AI doesn't drift into investment hype. Always fact-check output against [dgd/](../dgd/dgd-overview.md).

---

## The master guardrail block (paste at the top of any prompt)

```
COMPLIANCE RULES — never break, applies to script, captions, and on-screen text:
- Educational ONLY. This is not financial, investment, legal, or tax advice.
- Do NOT call Digital Gold (DGD) an investment.
- Do NOT predict or imply future price; the price curve is a distribution mechanism, not a forecast.
- Do NOT promise returns, profit, gains, or passive income.
- Do NOT say "buy," "get in early," "don't miss out," or solicit on the basis of profit.
- Describe IDEAS and DESIGN, never financial outcomes.
- Acceptable CTA: "follow to learn more" or "read the white paper" — never "buy."
```

## 1. Single explainer script

```
[PASTE master guardrail block]

ROLE: You script educational short-form videos for a Digital Gold (DGD) ambassador.
AUDIENCE: curious general public, comfortable on social media, no economics background.
TOPIC: [e.g., "Why money loses value over time"]
LENGTH: 40 seconds (~120 words spoken).
STRUCTURE: Hook (≤1s scroll-stopper) → curiosity gap → clear payoff (one idea) → loop-back CTA.
DELIVER as a table with columns: [Time] | [Spoken line] | [On-screen text] | [Suggested visual].
STYLE: short sentences, conversational, one idea per line, no jargon (define any term).
END with 5 alternative hooks for A/B testing and 5 compliant hashtags.
```

## 2. Hook generator

```
[PASTE master guardrail block]
Give me 15 scroll-stopping hooks (≤8 words each) for a short-form video about:
[TOPIC]. Use these frameworks: Contrarian Claim, Mistake Warning, List Tease, Shocking Number, Question. No banned words (invest, returns, moon, early). Curiosity only.
```

## 3. Six Pillars series (batch)

```
[PASTE master guardrail block]
Create a 6-part educational series, one video per "Pillar of Perfect Money":
Scarcity, Stable Pricing, Free Adoption, Decentralized Governance, Freedom to Transact, Adequate Circulation.
For EACH: a 35-second script (Hook → gap → payoff → loop), on-screen text, 3 visual ideas, and a compliant CTA. Keep a consistent intro/outro format so they feel like a series. Facts must be presentable as "how the design works," not profit.
```

## 4. Myth-buster

```
[PASTE master guardrail block]
Write a 30-second "myth → reality → why it matters" script debunking:
[MYTH, e.g., "more coins = more value"].
Make the reality surprising and shareable. Educational tone, no financial advice.
```

## 5. Explain-like-I'm-12

```
[PASTE master guardrail block]
Explain [CONCEPT, e.g., "the Crypto Fair Value formula"] for a 12-year-old in a 30-second script, using one everyday analogy. Accurate but simple. Frame as a measurement method, not a reason to buy.
```

## 6. Repurpose one script for 4 platforms

```
[PASTE master guardrail block]
Take this script: [PASTE].
Produce platform-tailored versions for TikTok, Instagram Reels, YouTube Shorts, and X:
- adjust length and caption,
- write a front-loaded disclosure line for each (I receive DGD ambassador recognition),
- give a compliant title for YouTube (searchable) and 3–5 hashtags per platform.
```

## 7. Fact-check pass (run AFTER drafting)

```
Review this DGD script for compliance and accuracy.
1) Flag ANY line that implies investment, profit, returns, price prediction, or solicitation.
2) Flag unverified claims/numbers and tell me which need source-checking against the white paper.
3) Rewrite flagged lines to be educational and design-focused.
Script: [PASTE]
```
